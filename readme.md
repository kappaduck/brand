# KappaDuck Brand

This repository contains the core styling for the KappaDuck brand.

## Usage

### 1. Include the CSS

Add the stylesheet to your HTML:

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/kappaduck/brand@main/kappaduck.min.css">
```

### 2. Load the Inter Font

```html
<link rel="preconnect" href="https://rsms.me/">
<link rel="stylesheet" href="https://rsms.me/inter/inter.css">
```
